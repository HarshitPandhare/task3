# Sales Data Analysis

## Project Description
The Sales Data Analysis project is a Python-based mini project that analyzes a sales dataset to extract meaningful business insights. The project uses the pandas library to process sales records and calculate important metrics such as total sales, best-selling product, and order statistics.

This project demonstrates basic data analysis concepts including data loading, cleaning, aggregation, and reporting.

---

## Objectives
- Load and analyze sales data from a CSV file
- Handle missing values appropriately
- Calculate total sales revenue
- Identify the best-selling product
- Compute multiple sales metrics
- Generate a simple sales analysis report

---

## Dataset
The dataset (`sales_data.csv`) contains sales records with the following columns:
- Date
- Product
- Quantity
- Price
- Customer_ID
- Region
- Total_Sales
